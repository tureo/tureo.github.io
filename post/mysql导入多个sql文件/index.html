需求

导入一个数据库的多个sql文件

存在问题

需要一个一个的执行导入sql文件语句

问题原因

mysql没有批量导入多个sql文件的命令

解决方法

windows

1.合并多个sql文件为一个文件 powershell执行命令get-content -Encoding utf8 *.sql > all.sql 参考

网上说的执行type *.sql > all.sql命令，会存在sql文件中文乱码问题，设置chcp 65001也无效，导致navicat等导入存在如下问题

linux

1.合并多个sql文件为一个文件

1

cat *.sql >> all.sql

2.单个文件写入多条sql语句 新建一个all.sql，vim all.sql，在里面写入需要执行的sql

1
2
3
4

source 1.sql
source 2.sql
source 53.sql
source 54.sql

登录mysql执行导入命令

source all.sql mysql> source all.sql
