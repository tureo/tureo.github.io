Github恢复被误删除的仓库

2021-10-26
github
约 212 字 预计阅读 1 分钟

文章目录

有时候不小心误删除github仓库，github官方提供一种恢复的方法，这里以恢复误删除的file仓库示例，操作如下：

  1. 登录GitHub，点击设置

  2. 点击仓库，已删除的仓库，恢复（注意这里的提示，删除的仓库可能需要一个小时左右才能看到，刚删除的仓库这个页面可能没显示出来，需要等一会；还有不能恢复fork的仓库）

  3. 确认恢复

  4. 等待恢复队列完成

  5. 完成恢复

  6. 已删除的仓库列表已没有file仓库了

  7. 回到自己的仓库列表，file仓库已恢复

参考官方文档
